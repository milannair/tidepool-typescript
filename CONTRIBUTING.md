# Contributing

Thanks for your interest in Tidepool TypeScript Client.

## How to contribute

- Open an issue to discuss a proposal before starting major work.
- Keep changes focused and include tests when behavior changes.
- Update `docs/tidepool-typescript-client-design.md` if the API contract changes.

## Development

```bash
npm install
npm run build
```

## Pull request checklist

- Clear description of what changed and why.
- Relevant docs updated.
- Tests added or a note explaining why they are not needed.

## Code of Conduct

This project follows the Contributor Covenant. See `CODE_OF_CONDUCT.md`.
